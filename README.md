# Bytecode Virtual Machine with Web Interface

## 🚴 Clone

```
git clone https://github.com/amalashkevich/vm-web.git
```

### 🛠️ Build with `wasm-pack build`

```
cd vm-web
wasm-pack build
```

### Install npm dependencies

```
cd www
npm i
```

### Run test web server

```
npm run start 
```

Open `http://localhost:8080` in the browser
